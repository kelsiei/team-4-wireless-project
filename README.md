# Security Threats and AI Defenses in 6G

Semester research project repository.

## Files
- `paper/main.tex` — main paper
- `paper/references.bib` — bibliography
- `notes/paper_summaries/` — individual paper summaries

## Research Focus
Our topic explores how AI changes the security landscape of 6G systems and what defense strategies are being proposed in current literature. The project pays particular attention to:
- adversarial machine learning threats in AI-enabled 6G
- O-RAN and wireless-specific attack scenarios
- data poisoning and clean-label poisoning defenses
- trust-aware and explainable security frameworks
- multilayer defense strategies for future 6G deployment
