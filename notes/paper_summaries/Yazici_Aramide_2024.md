Full Citation: [Oluwatosin Oladayo Aramide. Zero-trust identity principles in next-gen networks: AI-driven continuous verification for secure digital ecosystems. World Journal of Advanced Research and Reviews, 2024. https://wjarr.com/sites/default/files/fulltext_pdf/WJARR-2024-2656.pdf]

Summary

The paper explains that old cybersecurity methods (trusting users inside a network) don’t work well anymore. Instead, it uses Zero Trust, where every user and device must be checked all the time.
It also shows how AI can help by analyzing things like user behavior, location, and device info to decide if someone should get access. The system gives a trust score and allows or blocks access based on that. This helps stop threats like hackers using stolen accounts.

Relevance to Our Survey

This paper is useful because it shows how AI can be used for continuous security checks in modern networks. It explains how systems can automatically detect suspicious behavior and adjust access in real time, which is important for large and complex networks.

Critical Comment

A weakness is that it depends on collecting a lot of user data, which can cause privacy issues. Also, the system may be hard to explain and difficult to use with older systems.
