# Paper Summary - Cybersecurity for tactical 6G networks: Threats, architecture, and intelligence
Citation: Suomalainen, Jani, Ijaz Ahmad, Annette Shajan, and Tapio Savunen. "Cybersecurity for tactical 6G networks: Threats, architecture, and intelligence." Future Generation Computer Systems 162 (2025): 107500   

## Summary
This paper examines the security of "tactical bubbles," which are 6G networks that must operate reliably without on-site administrators. The authors propse a security architecture centered on Zero Trust principles, 
utilizing machine learning to perform autonomous monitoring and "zero-touch" management of network assets. To address the vulnerabilities of 6G intelligence, the paper maps specific adversarial tactics, such as poisoning 
training data and ML model evasion, to adaptive defensive strategies like adversarial input detection and behavior-based isolation.
                                                                                                   

## Relevance to our Survey
This directly addresses that 6G’s reliance on AI for critical tasks like resource allocation creates a major weakness susceptible to data poisoning. The paper's proposed architecture serves as a primary example of the "Zero 
Trust" defenses the project aims to investigate.

## An Issue I Identified
There is an issue with how there is not enough data for the AI to learn a baseline for what "normal" behaviour is. All attacks used to train to AI so far have been fake simulations. We don't know yet how the AI will perform 
in a real-word senario.
