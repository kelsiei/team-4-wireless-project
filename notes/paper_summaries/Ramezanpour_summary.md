# Paper Summary - Intelligent zero trust architecture for 5G/6G networks: Principles, challenges, and the role of machine learning in the context of O-RAN
  
  Citation: K. Ramezanpour and J. Jagannath, “Intelligent zero trust architecture for 5G/6G networks: Principles, challenges, 
  and the role of machine learning in the context of O-RAN,” Computer Networks, vol. 217, p. 109358, Sep. 2022, doi: https://doi.org/10.1016/j.comnet.2022.109358.
                                                                                                                                                                                                                  
  ## Summary

This paper introduces the concept of intelligent zero trust architecture (i-ZTA). It proposes a framework centered on three core functions (monitoring, evaluation, and decision-making) which use AI to perform real-time risk assessments for every individual access request. The architecture is specifically designed to be O-RAN compliant, utilizing the Near-Real-Time RAN Intelligent Controller (RIC) to host security-focused AI engines.
                                                                                                                                                                                                                  

  ## Relevance to our Survey

This source provides detailed specifics of AI methodologies such as Reinforcement Learning, Graph Neural Networks (GNN), and Federated Learning, which are proposed for 6G network defence.                                                                 

  ## An Issue I Identified

An Issue is a less trustworthy network perimeter of 6G, where functions run as virtualized software on shared third-party cloud infrastructure. This breaks the traditional assumption that everything inside the network can be trusted, and allows attackers to move laterally across the system once they have bypassed initial authentication.
