# Paper Summary - Securing the 6G–IoT Environment: A Framework for Enhancing Transparency in Artificial Intelligence Decision-Making Through Explainable Artificial Intelligence

**Citation:** [Kaur, N., & Gupta, L. (2025). Securing the 6G-IoT Environment: A Framework for Enhancing Transparency in Artificial Intelligence Decision-Making Through Explainable Artificial Intelligence. Sensors, 25(3), 854. https://doi.org/10.3390/s25030854]

## Summary

This paper explains how a security framework for IoT devices in upcoming 6G networks introduce serious vulnerabilities like eavesdropping, model poisoning and adversarial attacks. The author tests several machine learning models with real world IoT traffic datasets and found out that XGBoost(machine learning library) to be the best performer at 95% accuracy in classifying malicious versus normal traffic. The main idea is that making AI decisions more transparent helps build trust, especially in complex networks like 6G.

## Relevance

This is relevant to our topic because it shows how AI and machine learning can be used to detect security threats in 6G and IoT systems. The author argues that weakness in AI security systems is the lack of transparency which is explained using XAI to make threat detection decisions understandable to us. It also covers vulnerabilities like adversarial attacks and data poisoning which are key topics in our survey topic. 

## Critical Comment

One limitation is that there is no actual 6G data available yet, so the authors used a 5G IoT dataset and assumed it would be similar enough. It is hard to know if the results would hold up once real 6G networks are actually developed with new technologies like terahertz.
